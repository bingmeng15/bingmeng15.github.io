注册或者登录GitHub

点击查看 Github 教程

  1. 创建GitHub官网仓库

    • 首先我们前往官方网址：GitHub官网

    • 点击右上角的Sign up（注册），或者是Sign in（登录）,注册我就不说明了，不知道的可以自行百度。

  • 仓库的格式必须是：<用户名>.github.io

  • Description：为描述仓库（选填）

  • 勾选 Initialize this repository with a README 初始化一个 README.md 文件

  • 点击 Creat repository 创建

​ 那么，现在我们就可以进行下一步的步骤了

安装 Git 可跳过

点击查看 Git 配置教程
  1. 第一步安装好 Git
    • 点击鼠标右键,就可以看见已经安装好的、 Git ，或者在开始菜单搜索。
    • Git cmd是 Windows 命令行的指令风格。
    • Git Bash Linux 系统的指令风格（建议使用）
    • Git GUI 是图形化界面（新手学习不建议使用）
  2. 第二步，打开 Git Bash ，在创建好的博客目录下打开。
